Hello! This is a collaborative art project from Convox. We and assorted friends crafted a bunch of little guys, and let them go free to wander the world! We hope that we can all write their stories together. You can see all the guys out there on this map. If you find a guy, go to its page using the url code on its tag, and enter the code on its belly/body/body-equivalent, and then you can write the next chapter in that guy's adventure before leaving them somewhere to wander until they meet someone else! Hopefully some guys will get to travel the world (:
